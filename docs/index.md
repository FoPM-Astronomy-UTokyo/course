# IoA FoPM Practical Course
## 変革を駆動する先端物理・数学プログラム

[変革を駆動する先端物理・数学プログラム (FoPM)](https://www.s.u-tokyo.ac.jp/ja/FoPM/) は, 修士博士一貫の 5 年間の学位プログラムです. 基礎科学の専門人材に, 科学技術や社会イノベーションに広く影響を与えるためのスキルを提供することで, 彼らのポテンシャルを最大化するプログラムです.
